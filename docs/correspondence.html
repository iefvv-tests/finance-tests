Кореспонденція рахунків

Дебет / Кредит — стислий бюлетень для складання проведень

← На головну
Як зрозуміти будь-яке проведення без зубріння. У центрі — матриця Дебет/Кредит за типом рахунку: вона показує, по якій стороні рахунок зростає, а по якій зменшується. Далі — алгоритм, золота шпаргалка і таблиця типових проведень з поясненням логіки.

1. Головне правило

Дебет і кредит — це сторони рахунку, а не «плюс» чи «мінус».

Дебет і кредит самі по собі не означають «збільшення» чи «зменшення». Вони означають лише сторону рахунку (ліву — дебет, праву — кредит). Чи це «+», чи «−», залежить від типу рахунку:

Тип рахункуЗбільшення (▲)Зменшення (▼)
Ростуть по дебету
АктивиДебетКредит
ВитратиДебетКредит
Ростуть по кредиту
Зобов’язанняКредитДебет
Власний капіталКредитДебет
ДоходиКредитДебет
Дебет (Дт) — ліва сторона Кредит (Кт) — права сторона
Запам’ятай у два рядки:
Активи й витрати ростуть по дебету.
Зобов’язання, капітал і доходи ростуть по кредиту.

2. Алгоритм для будь-якої операції

Не питай одразу «який рахунок дебетувати?». Іди по кроках.
Що змінилось? Гроші, товари, борг постачальнику, дохід, витрати, зарплата, податок…
Це актив, зобов’язання, капітал, дохід чи витрата? Визнач тип кожного з двох рахунків, що беруть участь в операції.
Воно збільшилось чи зменшилось? Окремо для кожного рахунку: «▲ зросло» чи «▼ впало».
Постав Дт/Кт за матрицею. Тип рахунку + напрям зміни однозначно дають сторону (Дт або Кт).

3. Золота шпаргалка

Не ідеальне правило, але закриває ~70% задач.
Що отримали / куди прийшло — у Дебет.
Звідки взялося / джерело — у Кредит.
Отримали гроші в банку (овердрафт / короткострокова позика)
Гроші прийшли → Дт 31 · джерело — позика → Кт 60 ⟶ Дт 31 Кт 60
Актив (гроші) зріс — по дебету; зобов’язання (позика) виникло — по кредиту.
Безоплатно одержали виробничі запаси
Запаси прийшли → Дт 20 · джерело — дохід → Кт 718 ⟶ Дт 20 Кт 718
Актив (запаси) зріс — по дебету; визнано дохід — по кредиту.

4. Швидкі правила-якорі

Те, що майже завжди працює без роздумів.
Витрати майже завжди → у Дебет
Нарахували зарплату виробничникам: витрати ▲ → Дт 23, борг перед працівниками ▲ → Кт 661 ⟶ Дт 23 Кт 661
Будь-яке зростання витрат (виробництво, збут, адміністрація) — це дебет.
Доходи майже завжди → у Кредит
Продали товар у борг: дебіторка ▲ → Дт 36, дохід ▲ → Кт 702 ⟶ Дт 36 Кт 702
Одночасно списують собівартість: витрати ▲ → Дт 902, товари ▼ → Кт 28 ⟶ Дт 902 Кт 28
Дохід визнають по кредиту, а собівартість проданого — окремим проведенням по дебету.
Переміщення всередині активів
Готову продукцію оприбуткували з виробництва: ГП ▲ → Дт 26, виробництво ▼ → Кт 23 ⟶ Дт 26 Кт 23
Один актив зріс (дебет), інший на ту саму суму зменшився (кредит). Валюта балансу не змінюється.

7. Суперкоротка формула для іспиту

Постав собі 3 питання: що ▲ зросло? що ▼ впало (за рахунок чого)? який це тип рахунку?
Актив ▲ = Дт
Актив ▼ = Кт
Зобов’язання ▲ = Кт
Зобов’язання ▼ = Дт
Капітал ▲ = Кт
Дохід ▲ = Кт
Витрати ▲ = Дт
Не вчи проведення списком. Натренуй 20–30 типових операцій — і далі воно почне «клацати» автоматично.
